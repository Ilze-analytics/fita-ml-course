# ML kurss — fita-ml-course

Šis repozitorijs satur manus mājas darbus
kursam **"Machine Learning un prognozēšanas modeļi"**.

## Kursa mērķis

Attīstīt prasmes izmantot gatavus mašīnmācīšanās algoritmus un metodes praktisku datu problēmu risināšanai, 
sagatavot datus modeļu treniņam un interpretēt rezultātus.

## Izmantotās tehnoloģijas

* Python
* Jupyter Notebook
* pandas
* matplotlib
* seaborn
* scikit-learn

## Kursa tēmas

1. **Ievads ML**
2. **Datu sagatavošana**
3. **Klasifikācija**
4. **Regresija**
5. **Klasterizācija un interpretācija**
6. **Gala projekts**

## Repozitorija struktūra

* `week1/` — ievads mašīnmācīšanās metodēs
* `week2/` — datu sagatavošana
* `week3/` — klasifikācijas modeļi
* `week4/` — regresijas modeļi
* `week5/` — klasterizācija un interpretācija
* `final-project/` — gala projekts
